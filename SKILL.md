# Scan to Skill

Install skills from QR codes in one flow: decode -> parse slug -> install -> verify.

## Auto Trigger (Telegram image -> auto install)

When a user sends an image and asks to install from QR, run this skill automatically:

1. Detect image attachment in the current message.
2. Pass the local image path to `scripts/install_from_qr.py`.
3. Install immediately when payload resolves to a valid slug.
4. Reply with decoded text, parsed slug, and install result.

## Workflow

1. Get QR input image
   - Use uploaded image, screenshot, or camera snapshot.
2. Decode QR content
   - Prefer `scripts/install_from_qr.py --decode-only <image>`.
3. Parse install target
   - Accept:
     - direct slug (`skill-feed`)
     - ClawHub URL (`https://clawhub.ai/<owner>/<slug>` or `https://clawhub.ai/<slug>`)
     - install command text (`clawhub install <slug>`)
4. Install with ClawHub CLI
   - Run install command generated by script.
5. Verify installation
   - Check with `clawhub list`.

## Commands

- Decode only:
  - `python3 scripts/install_from_qr.py --decode-only <image_path>`
- Decode + install:
  - `python3 scripts/install_from_qr.py <image_path>`
- Install to custom dir:
  - `python3 scripts/install_from_qr.py <image_path> --dir skills`

## Safety

- Only install if decoded content resolves to a valid slug.
- If QR decodes to non-ClawHub URL, do not auto-install; ask for confirmation.
- For unknown or suspicious slugs, recommend vetting before install.

## Output format

- Decoded QR: `<raw_text>`
- Parsed slug: `<slug>`
- Install command: `<command>`
- Result: success/failure + reason
- Next step: open skill page or run quick test
